# 学生功能开发指南

> 本文档告诉你：**开发一个功能页面需要做什么、怎么用已有的基础设施、哪些东西不能碰。**

---

## 一、学生只需要创建两个文件

```
oaepp/pages/xxx.py        ← 页面组件（UI）
oaepp/states/xxx.py       ← 功能状态（数据逻辑）
```

**文件命名必须与 prototype 目录下的原型文件一致：**

| 原型文件 | 你要创建的页面文件 | 你要创建的状态文件 |
|----------|-------------------|-------------------|
| `dashboard.html` | `pages/dashboard.py` | `states/dashboard.py` |
| `courses.html` | `pages/courses.py` | `states/courses.py` |
| `assignments.html` | `pages/assignments.py` | `states/assignments.py` |
| `grades.html` | `pages/grades.py` | `states/grades.py` |
| `attendance.html` | `pages/attendance.py` | `states/attendance.py` |
| `exam.html` | `pages/exam.py` | `states/exam.py` |
| `profile.html` | `pages/profile.py` | `states/profile.py` |

创建后路由自动生效，**不需要修改 `app.py`**。

## 二、标准模板

### `pages/xxx.py` — 页面组件

```python
"""成绩看板 (F-S-030)"""
import reflex as rx
from oaepp.components.layout import page_layout
from oaepp.components.common import stat_card, empty_state, loading_spinner
from oaepp.states.grades import GradesState


def grades_page():
    """成绩看板页面 — 函数名必须是 {模块名}_page"""
    return page_layout(
        title="成绩看板",
        content=rx.vstack(
            # 统计卡片
            rx.grid(
                stat_card("总成绩", "85.5", icon="bar_chart_3"),
                stat_card("排名", "12/45", icon="trophy"),
                columns="2",
                spacing="4",
                width="100%",
            ),
            # 加载状态
            rx.cond(
                GradesState.loading,
                loading_spinner(),
                rx.box(),
            ),
            # 空状态
            rx.cond(
                GradesState.grades.length() == 0,
                empty_state("暂无成绩数据"),
                # 数据列表
                rx.foreach(GradesState.grades, _grade_row),
            ),
            spacing="4",
            width="100%",
        ),
    )


def _grade_row(grade):
    """单行成绩"""
    return rx.card(
        rx.hstack(
            rx.text(grade["course_name"]),
            rx.text(grade["score"]),
            justify="between",
            width="100%",
        ),
    )
```

### `states/xxx.py` — 功能状态

```python
"""成绩状态管理 (F-S-030)"""
import reflex as rx
from typing import List
from sqlmodel import select
from oaepp.models import ScoreItem, Course


class GradesState(rx.State):
    """成绩状态 — 独立 State，不继承 GlobalState"""
    grades: List[dict] = []
    loading: bool = False
    error_message: str = ""

    async def load_grades(self):
        """从数据库加载成绩"""
        self.loading = True
        self.error_message = ""
        try:
            with rx.session() as session:
                items = session.exec(
                    select(ScoreItem, Course)
                    .join(Course, ScoreItem.course_id == Course.id)
                    .where(ScoreItem.student_user_id == 1)  # TODO: 从 AuthState 获取
                ).all()
                self.grades = [
                    {
                        "course_name": course.name,
                        "score_type": item.score_type,
                        "score": str(item.score),
                    }
                    for item, course in items
                ]
        except Exception as e:
            self.error_message = f"加载失败: {e}"
        finally:
            self.loading = False
```

## 三、学生可以使用的基础设施

### 1. ORM 模型（查数据库）

```python
from sqlmodel import select
from oaepp.models import User, Course, Enrollment, Assignment, Submission

# 在 State 方法中使用
with rx.session() as session:
    # 查询
    courses = session.exec(select(Course)).all()
    
    # 条件查询
    my_courses = session.exec(
        select(Course)
        .join(Enrollment)
        .where(Enrollment.student_user_id == 1)
    ).all()
    
    # 写入
    new_item = Assignment(title="作业1", course_id=1)
    session.add(new_item)
    session.commit()
```

**可用模型（全部从 `oaepp.models` 导入）：**

| 模型 | 对应表 | 常用场景 |
|------|--------|---------|
| `User` | users | 登录、个人资料 |
| `Course` | courses | 课程列表 |
| `Chapter` | chapters | 章节展示 |
| `Enrollment` | enrollments | 我的课程 |
| `Assignment` | assignments | 作业列表 |
| `Submission` | submissions | 作业提交 |
| `Exam` | exams | 在线考试 |
| `ExamQuestion` | exam_questions | 考试题目 |
| `ExamAttempt` | exam_attempts | 考试答卷 |
| `AttendanceSession` | attendance_sessions | 签到场次 |
| `AttendanceRecord` | attendance_records | 签到记录 |
| `ScoreItem` | score_items | 成绩条目 |
| `GradingRecord` | grading_records | 批改记录 |
| `Notification` | notifications | 通知消息 |

> 完整模型定义见 `oaepp/models/database.py`

## 四、完整开发流程

```bash
# 1. 拉取最新代码
git checkout main && git pull

# 2. 创建功能分支
git checkout -b feature/你的姓名-功能名

# 3. 创建两个文件
#    oaepp/pages/grades.py
#    oaepp/states/grades.py

# 4. 本地运行测试
cd oaepp/
reflex run
# 浏览器访问 http://localhost:3000/grades

# 5. 提交并推送
git add oaepp/pages/grades.py oaepp/states/grades.py
git commit -m "feat(grades): 实现成绩看板页面 F-S-030"
git push origin feature/你的姓名-功能名

# 6. 在 GitHub 上发起 PR，目标分支选 main
```

## 五、禁止修改的文件（出现即打回）

| 文件/目录 | 说明 |
|----------|------|
| `oaepp/models/` | ORM 模型层，负责人维护 |
| `oaepp/components/` | 共享 UI 组件，负责人维护 |
| `oaepp/states/__init__.py` | State 注册文件 |
| `oaepp/rxconfig.py` | Reflex 配置 |
| `oaepp/database.py` | 数据库连接池 |
| `oaepp/constants.py` | 全局常量 |
| `oaepp/utils/` | 工具函数 |
| `oaepp/app.py` | 路由注册中心 |
| `backend/` | FastAPI 后端（与 Reflex 无关） |

**你只能创建和修改：**
- `oaepp/pages/你的功能.py`
- `oaepp/states/你的功能.py`
- `tests/reflex/你的测试.py`（可选）

## 六、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `ModuleNotFoundError: No module named 'oaepp.models'` | 未在 oaepp 目录下运行 | `cd oaepp && reflex run` |
| 路由 404 | 函数名不是 `{模块名}_page` | 检查函数命名，如 `grades_page` |
| `rx.session()` 报错 "No database url configured" | 未在 oaepp 目录下运行 | `cd oaepp && reflex run` |
| 侧边栏没有高亮当前页面 | 路由路径不匹配 | 确认 `route="/grades"` 与 `_NAV_ITEMS` 中一致 |
| 导入 `from oaepp.models import ...` 失败 | sqlmodel 未安装 | `pip install sqlmodel` |
