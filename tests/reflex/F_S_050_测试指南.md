### 3.2 数据库自动重连测试

**前置条件：** 需要 Python 交互环境

```bash
cd D:\123456\123\OA-EPP-main\OA-EPP-main
python
```

```python
import asyncio
import sys
sys.path.insert(0, '.')

async def test_reconnect():
    from oaepp.database import check_connection, db, close_async_pool

    # 1. 正常连接检查
    ok = await check_connection()
    print(f"[1] 连接健康检查: {'PASS' if ok else 'FAIL'}")

    # 2. 执行查询
    async with db() as cur:
        await cur.execute("SELECT 1 AS val")
        row = await cur.fetchone()
        print(f"[2] 查询测试: {'PASS' if row['val'] == 1 else 'FAIL'}")

    # 3. 关闭连接池后重新连接（模拟断连恢复）
    await close_async_pool()
    print("[3] 连接池已关闭（模拟断连）")

    ok2 = await check_connection()
    print(f"[4] 断连后自动重连: {'PASS' if ok2 else 'FAIL'}")

    # 5. 重连后可正常查询
    async with db() as cur:
        await cur.execute("SELECT VERSION() AS ver")
        row = await cur.fetchone()
        print(f"[5] 重连后查询: PASS (MySQL {row['ver']})")

    await close_async_pool()
    print("\n✅ 全部数据库韧性测试通过")

asyncio.run(test_reconnect())
```

**验收标准：**

| # | 步骤 | 预期结果 |
|---|------|----------|
| 1 | 初始连接健康检查 | PASS |
| 2 | 执行 SQL 查询 | PASS，返回 val=1 |
| 3 | 关闭连接池 | 日志显示 "已关闭" |
| 4 | 自动重连 | PASS，日志显示重试过程 |
| 5 | 重连后查询 | PASS，返回 MySQL 版本号 |
